# DativeJS

## What is DativeJS?

DativeJS is a modern reactive UI JavaScript framework that brings powerful reactivity, component-based architecture, and elegant templating to your web applications. Built with developer experience in mind, DativeJS provides an intuitive API that makes building dynamic interfaces both simple and enjoyable.

With DativeJS, you write declarative templates with reactive data binding, and the framework automatically updates your UI when data changes. No need for virtual DOM diffing or manual DOM manipulation—DativeJS handles it all efficiently.

## Quick example

Here's a simple counter application to give you a taste of DativeJS:

```javascript
import Dative from 'dativejs';

const app = new Dative({
  el: '#app',
  data() {
    return {
      count: 0
    }
  },
  methods: {
    increment() {
      this.count++;
    }
  },
  template: `
    <div>
      <h1>Counter: {{ count }}</h1>
      <button dv-on:click="increment()">Increment</button>
    </div>
  `
});
```

That's all you need! DativeJS will automatically track the `count` data and update the UI when it changes.

## Why DativeJS?

### Simplicity first

DativeJS is designed to be easy to learn and use. If you know JavaScript, HTML, and CSS, you're ready to build with DativeJS. No complex build tools required—though it works great with modern bundlers too.

### Lightweight and performant

At its core, DativeJS is lightweight and uses efficient DOM diffing algorithms to minimize DOM operations. Your applications stay fast even as they grow in complexity.

### Flexible and extensible

Whether you're building a simple widget or a complex single-page application, DativeJS scales with your needs. The plugin system and custom directives let you extend the framework exactly how you need.
