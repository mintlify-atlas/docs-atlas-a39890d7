This guide will walk you through creating a simple todo list application with DativeJS, demonstrating the core features of the framework.

## Prerequisites

Make sure you have:

- Node.js 14+ installed (if using npm)
- A text editor
- Basic knowledge of JavaScript, HTML, and CSS

## Create your first app

<Steps>
  <Step title="Set up your project">
    Create a new directory and initialize a project:

    ```bash
    mkdir my-dative-app
    cd my-dative-app
    npm init -y
    npm install dativejs
    ```

    Create an `index.html` file:

    ```html
    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>My Todo App</title>
      <style>
        body {
          font-family: system-ui, -apple-system, sans-serif;
          max-width: 600px;
          margin: 40px auto;
          padding: 20px;
        }
        [dv-cloak] { display: none; }
      </style>
    </head>
    <body>
      <div id="app"></div>
      <script type="module" src="./app.js"></script>
    </body>
    </html>
    ```
  </Step>

  <Step title="Create the application">
    Create an `app.js` file with your DativeJS application:

    ```javascript
    import Dative from 'dativejs';

    const app = new Dative({
      el: '#app',
      
      data() {
        return {
          todos: [
            { id: 1, text: 'Learn DativeJS', completed: false },
            { id: 2, text: 'Build an app', completed: false }
          ],
          newTodo: '',
          filter: 'all' // 'all', 'active', 'completed'
        }
      },

      computed: {
        filteredTodos() {
          if (this.filter === 'active') {
            return this.todos.filter(todo => !todo.completed);
          }
          if (this.filter === 'completed') {
            return this.todos.filter(todo => todo.completed);
          }
          return this.todos;
        },
        
        remainingCount() {
          return this.todos.filter(todo => !todo.completed).length;
        }
      },

      methods: {
        addTodo() {
          if (this.newTodo.trim()) {
            this.todos.push({
              id: Date.now(),
              text: this.newTodo.trim(),
              completed: false
            });
            this.newTodo = '';
          }
        },

        toggleTodo(id) {
          const todo = this.todos.find(t => t.id === id);
          if (todo) {
            todo.completed = !todo.completed;
            this.render();
          }
        },

        removeTodo(id) {
          const index = this.todos.findIndex(t => t.id === id);
          if (index > -1) {
            this.todos.splice(index, 1);
            this.render();
          }
        },

        setFilter(filter) {
          this.filter = filter;
        }
      },

      template: `
        <div dv-cloak>
          <h1>Todo List</h1>
          
          <div style="margin-bottom: 20px;">
            <input 
              type="text" 
              bind:value="newTodo"
              dv-on:keyup="if ($event.key === 'Enter') addTodo()"
              placeholder="What needs to be done?"
              style="padding: 8px; width: 70%; font-size: 16px;"
            />
            <button 
              dv-on:click="addTodo()"
              style="padding: 8px 16px; margin-left: 8px; font-size: 16px;"
            >
              Add
            </button>
          </div>

          <div style="margin-bottom: 20px;">
            <button 
              dv-on:click="setFilter('all')"
              style="padding: 6px 12px; margin-right: 8px;"
            >
              All ({{ todos.length }})
            </button>
            <button 
              dv-on:click="setFilter('active')"
              style="padding: 6px 12px; margin-right: 8px;"
            >
              Active ({{ remainingCount }})
            </button>
            <button 
              dv-on:click="setFilter('completed')"
              style="padding: 6px 12px;"
            >
              Completed ({{ todos.length - remainingCount }})
            </button>
          </div>

          #if filteredTodos.length > 0
            <ul style="list-style: none; padding: 0;">
              #each filteredTodos as todo
                <li style="
                  padding: 12px;
                  margin-bottom: 8px;
                  background: #f5f5f5;
                  border-radius: 4px;
                  display: flex;
                  align-items: center;
                  gap: 12px;
                ">
                  <input 
                    type="checkbox" 
                    bind:checked="todo.completed"
                    dv-on:change="toggleTodo({{ todo.id }})"
                  />
                  <span style="
                    flex: 1;
                    {{ todo.completed ? 'text-decoration: line-through; color: #888;' : '' }}
                  ">
                    {{ todo.text }}
                  </span>
                  <button 
                    dv-on:click="removeTodo({{ todo.id }})"
                    style="padding: 4px 12px; background: #ff4444; color: white; border: none; border-radius: 4px; cursor: pointer;"
                  >
                    Delete
                  </button>
                </li>
              /each
            </ul>
          :else
            <p style="color: #888; text-align: center; padding: 40px;">
              #if filter === 'completed'
                No completed todos yet
              :else if filter === 'active'
                No active todos
              :else
                No todos yet. Add one above!
              /if
            </p>
          /if

          #if todos.length > 0
            <p style="margin-top: 20px; color: #666;">
              {{ remainingCount }} {{ remainingCount === 1 ? 'item' : 'items' }} remaining
            </p>
          /if
        </div>
      `
    });
    ```
  </Step>

  <Step title="Run your application">
    You can use a simple development server to run your app. If you have Python installed:

    ```bash
    # Python 3
    python -m http.server 8000
    ```

    Or use a modern dev server like Vite:

    ```bash
    npx vite
    ```

    Open your browser to `http://localhost:8000` (or the port shown by your dev server).
  </Step>

  <Step title="Test the features">
    Your todo app is now running! Try:

    - Adding new todos by typing in the input and clicking **Add** or pressing Enter
    - Toggling todos as completed by clicking the checkboxes
    - Deleting todos with the **Delete** button
    - Filtering todos using the **All**, **Active**, and **Completed** buttons

    Notice how the UI updates automatically when you interact with it—that's DativeJS reactivity in action!
  </Step>
</Steps>

## What's happening?

Let's break down the key concepts:

### Reactive data

The `data()` function returns an object that becomes reactive. When you modify properties like `todos` or `newTodo`, DativeJS automatically updates the DOM:

```javascript
data() {
  return {
    todos: [],
    newTodo: ''
  }
}
```

### Template syntax

DativeJS templates use double curly braces for interpolation and special directives for control flow:

- `{{ expression }}` - Interpolate JavaScript expressions
- `#if condition` / `:else` / `/if` - Conditional rendering
- `#each array as item` / `/each` - Loop through arrays
- `dv-on:event="handler()"` - Event handling
- `bind:attribute="data"` - Bind element attributes to data

### Methods

Methods are functions that can access and modify your reactive data using `this`:

```javascript
methods: {
  addTodo() {
    this.todos.push({ /* new todo */ });
  }
}
```

### Computed properties

Computed properties are derived values that automatically update when their dependencies change:

```javascript
computed: {
  remainingCount() {
    return this.todos.filter(todo => !todo.completed).length;
  }
}
```

## Advanced example

Ready for more? Here's how to add animations to your todo app:

```javascript
template: `
  <li animate:fadein="200,0" style="...">
    <!-- todo content -->
  </li>
`
```

The `animate:fadein` directive applies a fade-in animation with a 200ms duration and 0ms delay.

Learn more about animations in the [animations guide](/guides/animations).
